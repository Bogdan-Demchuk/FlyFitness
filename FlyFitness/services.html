Наші послуги

Реальні напрямки тренувань у FlyFitness GYM. Тільки професійне обладнання, профільні бійцівські секції та кваліфікований тренерський супровід.

Тренажерний зал та Кардіо

Зона для самостійних занять, що включає силові тренажери на всі групи м'язів, великий вибір вільних ваг (гантелі, штанги) та зону кардіо-обладнання для зміцнення серцево-судинної системи.

Доступні варіанти абонементів на фіксовану кількість занять (від 8 до 25), довгострокові безліміти, а також вигідний Денний безліміт для тих, хто тренується з 11:00 до 15:00.

  • ✔ Самостійні тренування без обмежень за часом
  • ✔ Професійна кардіо-зона
  • ✔ Спеціальні умови для денного відвідування
  • ✔ Можливість разових візитів

Групові бойові тренування

Спеціалізовані секції з єдиноборств для дорослих під керівництвом досвідчених інструкторів. Заняття проходять у групах та спрямовані на освоєння техніки, розвиток витривалості та координації.

У клубі діють окремі профільні класи з Боксу, Бразильського Джиу-Джитсу (БДД) та Кікбоксингу, включаючи окремі Жіночі групи з кікбоксингу.

  • ✔ Класичний Бокс та Бразильське Джиу-Джитсу
  • ✔ Загальні групи з Кікбоксингу
  • ✔ Спеціалізований Жіночий Кікбоксинг
  • ✔ Можливість окремого або сумісного відвідування залів

Персональний та VIP тренінг

Заняття під прямим контролем тренера для тих, кому потрібен індивідуальний підхід, правильна техніка виконання вправ та швидкий результат без травм.

Ви можете обрати доступніші тренування у міні-групах під наглядом штатного чи Старшого тренера, або ж скористатися преміальним форматом Індивідуальних VIP-тренувань 1 на 1.

  • ✔ Тренування у міні-групах з тренером
  • ✔ Заняття з кваліфікованим Старшим тренером
  • ✔ Індивідуальний VIP-формат (повністю сам на сам з наставником)
  • ✔ Контроль техніки, безпеки та індивідуальне навантаження
